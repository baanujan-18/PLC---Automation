# Demonstration Video

This video demonstrates the PLC-based metal-sorting conveyor system.

The demonstration includes:

- Object detection using the D1 photoelectric sensor
- Two-second delayed conveyor startup
- Metal detection using the D2 inductive proximity sensor
- Automatic conveyor stopping
- Pneumatic ejection of the metallic object
- Non-metallic object movement toward the D3 photoelectric sensor
- Green, orange, and red status indications
